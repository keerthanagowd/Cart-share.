# CartShare - Collaborative Shopping Ecosystem

**12052026_KeerthanaGowd_CartShare** | Frontend Internship Project

## 📋 Project Overview
CartShare is a real-time collaborative web app designed to solve communication gaps during group online shopping. Users can create/join rooms with unique codes,manage a shared cart with real-time updates for all users in the room.

Built for hostel/office scenarios where friends order together from Swiggy, Blinkit, Zepto, etc.

- **Room System**: Create or join shopping rooms using `?room=` URL code
- **Real-time Sync**: Add or delete items with instant updates for all users in room
- **Data Persistence**: Uses Firebase Realtime Database for sync across devices

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, Bootstrap 5
- **Logic**: Vanilla JavaScript ES6+
- **Styling**: Custom CSS with Flexbox & CSS Grid for responsive design
- **Storage**: Firebase Realtime Database
