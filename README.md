# CartShare - Collaborative Shopping Ecosystem

**12052026_KeerthanaGowd_CartShare** | Frontend Internship Project

## 📋 Project Overview
CartShare is a real-time collaborative web app designed to solve communication gaps during group online shopping. Users can create/join rooms with unique codes, manage a shared cart, track activities, and generate split receipts for easy expense settlement.

Built for hostel/office scenarios where friends order together from Swiggy, Blinkit, Zepto, etc.

## ✨ Features Implemented
- **Room System**: Create or join shopping rooms using unique codes on the landing page
- **Shared Dashboard**: Real-time collaborative cart where all users can add/modify items
- **Item Management**: Add products with name, quantity, and price with instant updates
- **Activity Log**: Tracks all actions with user names and timestamps for transparency
- **Smart Calculations**: Auto-calculates subtotal, taxes, delivery charges, and per-person split
- **Receipt Generation**: Clean, printable receipts for expense settlement
- **Data Persistence**: Uses Browser LocalStorage to maintain room data across sessions

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, Bootstrap 5
- **Logic**: Vanilla JavaScript ES6+
- **Styling**: Custom CSS with Flexbox & CSS Grid for responsive design
- **Storage**: LocalStorage API for client-side data persistence
